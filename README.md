# Ashar Shamim — Portfolio

Built with [Vite](https://vitejs.dev) · Deployed on [Vercel](https://vercel.com)

## Local development

```bash
npm install      # first time only
npm run dev      # starts at http://localhost:5173
```

Edit `index.html`, `style.css`, or `main.js` and the browser updates instantly.

## Adding assets

Put images in `public/images/`, PDFs in `public/pdfs/`, certificates in `public/certificates/`.
Reference them in HTML as `/images/file.png` (no `public` prefix needed).

## Deploy

```bash
git add .
git commit -m "your message"
git push
```

Vercel auto-deploys on every push to `main`.
